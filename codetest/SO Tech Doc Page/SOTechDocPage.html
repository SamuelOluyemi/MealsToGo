Introduction

JavaScript est un langage de script multiplateforme et orienté objet. Il s'agit d'un langage petit et léger. À l'intérieur d'un environnement hôte (par exemple, un navigateur Web), JavaScript peut être connecté aux objets de son environnement pour fournir un contrôle programmatique sur ceux-ci.

JavaScript contient une bibliothèque standard d'objets, tels que Array, Date et Math, et un ensemble d'éléments de langage de base tels que les opérateurs, les structures de contrôle et les instructions. Le JavaScript de base peut être étendu à des fins diverses en le complétant par des objets supplémentaires, par exemple :

  • Le JavaScript côté client étend le langage de base en fournissant des objets pour contrôler un navigateur et son Document Object Model (DOM). Par exemple, les extensions côté client permettent à une application de placer des éléments sur un formulaire HTML et de répondre aux événements de l'utilisateur tels que les clics de souris, la saisie de formulaires et la navigation dans la page.
  • Le JavaScript côté serveur étend le langage de base en fournissant des objets pertinents pour l'exécution de JavaScript sur un serveur. Par exemple, les extensions côté serveur permettent à une application de communiquer avec une base de données, d'assurer la continuité des informations d'une invocation à l'autre de l'application ou d'effectuer des manipulations de fichiers sur un serveur.
Ce qu'il faut déjà savoir

Ce guide suppose que vous disposez des connaissances de base suivantes :

  • Une compréhension générale de l'Internet et du World Wide Web (WWW).
  • Bonne connaissance pratique du langage de balisage hypertexte (HTML).
  • Une certaine expérience de la programmation. Si vous êtes novice en programmation, essayez l'un des didacticiels liés à la page principale sur JavaScript.

Salut le Monde
Pour commencer à écrire du JavaScript, ouvrez le Scratchpad et écrivez votre premier code JavaScript "Salut le Monde" : function salueMoi(tonNom) { alerte("Salut " + tonNom); } salueMoi("leMonde"); Sélectionnez le code dans le pad et appuyez sur Ctrl+R pour le voir se dérouler dans votre navigateur !
